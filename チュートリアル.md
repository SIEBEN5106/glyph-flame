
## インストール
GlyphFlameはフォーク元のOceanFlameと異なりオンラインツールを用意していません。
利用する場合はリポジトリをダウンロードしてローカルで利用してください。


### 必要なもの
* Bun(事前に導入しておいてください)
* glyph-flame-main(リポジトリのコピー)
* bdfフォントファイル(フォントを置き換える場合)


1. GlyphFlameのリポジトリで`Code > Download ZIP`を押してリポジトリをダウンロードする。
2. ダウンロードしたZIPを任意の場所に解凍し、`glyph-flame-main`のフォルダ直下にリポジトリの中身が入っているかを確認する。
```
.
└── glyph-flame-main/
    └── (リポジトリの中身)
```
3. `glyph-flame-main`のフォルダを`Shift+右クリック`して`Powershell ウィンドウをここで開く`を押す。
4. `bun install`で依存関係をインストールする。



## 使い方

1. `glyph-flame-main`のフォルダを`Shift+右クリック`して`Powershell ウィンドウをここで開く`を押す。
2. `bun run dev`で開発サーバーを起動し`O+Enter`キーを押す。
3. 編集したい`.img`ファイルを、画面の中央付近にドラッグ&ドロップする。
4. 左側のツリーから編集したいフォントの種類(SMALL/LARGE)とFont Plane(フォント平面)を選択する。
(例：`Hiragana`、`Katakana`、`CJK Unified`など)

### 選択中のフォント平面にBDFフォントを読み込ませる
5. 対応するサイズの`.bdf`ファイルを、画面中央のエリアにドラッグ&ドロップする。
* Smallフォント → 12px 程度のBDF（`efont-unicode-bdf`の`b12.bdf`など）
* Largeフォント → 16px 程度のBDF（`efont-unicode-bdf`の`b16.bdf`など）

※ 初回は反映されない場合が多いです。その場合は一旦別の平面に移動してから、再度目的の平面に戻り、BDFをもう一度ドラッグ&ドロップしてください。

※ `Hiragana`や`Katakana`はbdfフォントをインポートしただけでは、実機で右端が途切れて表示される場合がある。
対策として、`Align All Left`で選択中のフォント平面の全文字を左詰めする。

※ フォント平面を移動する前に、`Apply Small / Apply Large`を押してその平面の編集を保存してください。


### 文字の編集
6. 調整したい文字をクリックして選択すると、下側に編集ウィンドウが開く。

#### 編集方法
* クリック → ドットのON/OFF
* ドラッグ → 範囲選択
* 矢印キー → 選択範囲の移動
* Escキー または 「Clear Selection」 → 範囲選択を解除

7. 編集が終わったら 「Apply This Glyph」 をクリックしてその文字の変更を確定する。

### 保存
8. すべての編集が完了したら 「Download .img」 をクリックして、編集済みのファームウェアを保存する。

### かなと漢字を置き換える場合
自分の場合は以下のフォント平面を編集しています。（SMALL/LARGE両方）

* 「Apply This Glyph」を行う
 * Hiragana
 * Katakana
 * CJK_Unified

* 「Apply This Glyph」は行わずぞのまま
 * CJK_Compatibility
 * CJK_Extention_A
 * CJK_Compatibility_Ideographs
