
## インストール
GlyphFlameはフォーク元のOceanFlameと異なりオンラインツールを用意していません。<br>
利用する場合はリポジトリをダウンロードしてローカルで利用してください。

Scroll down for the English Tutorial



#### 必要なもの
* Bun(事前に導入しておいてください)
* glyph-flame-main(リポジトリのコピー)
* bdfフォントファイル(フォントを置き換える場合)


1. GlyphFlameのリポジトリで`Code > Download ZIP`を押してリポジトリをダウンロードする。
2. ダウンロードしたZIPを任意の場所に解凍し、`glyph-flame-main`のフォルダ直下にリポジトリの中身が入っているかを確認する。
```
.
└── glyph-flame-main/
    └── (リポジトリの中身)
```
3. `glyph-flame-main`のフォルダを`Shift+右クリック`して`Powershell ウィンドウをここで開く`を押す。
4. `bun install`で依存関係をインストールする。



## 使い方

1. `glyph-flame-main`のフォルダを`Shift+右クリック`して`Powershell ウィンドウをここで開く`を押す。
2. `bun run dev`で開発サーバーを起動し`O+Enter`キーを押す。
3. 編集したい`.img`ファイルを、画面の中央付近にドラッグ&ドロップする。
4. 左側のツリーから編集したいフォントの種類(SMALL/LARGE)とFont Plane(フォント平面)を選択する。
(例：`Hiragana`、`Katakana`、`CJK Unified`など)

### 選択中のフォント平面にBDFフォントを読み込ませる
5. 対応するサイズの`.bdf`ファイルを、画面中央のエリアにドラッグ&ドロップする。
* Smallフォント → 12px 程度のBDF（`efont-unicode-bdf`の`b12.bdf`など）
* Largeフォント → 16px 程度のBDF（`efont-unicode-bdf`の`b16.bdf`など）

※ 初回は反映されない場合が多いです。その場合は一旦別の平面に移動してから、再度目的の平面に戻り、BDFをもう一度ドラッグ&ドロップしてください。

※ `Hiragana`や`Katakana`はbdfフォントをインポートしただけでは、実機で右端が途切れて表示される場合があります。<br>
対策として、`Align All Left`で選択中のフォント平面の全文字を左詰めしてください。

※ フォント平面を移動する前に、`Apply Small / Apply Large`を押してその平面の編集を保存してください。


### 文字の編集
6. 調整したい文字をクリックして選択すると、下側に編集ウィンドウが開く。

#### 編集方法
* クリック → ドットのON/OFF
* ドラッグ → 範囲選択
* 矢印キー → 選択範囲の移動
* Escキー または 「Clear Selection」 → 範囲選択を解除

7. 編集が終わったら 「Apply This Glyph」 をクリックしてその文字の変更を確定する。

### 保存
8. すべての編集が完了したら 「Download .img」 をクリックして、編集済みのファームウェアを保存する。

  * Hiragana
  * Katakana
  * CJK_Unified

  * CJK_Compatibility

  * CJK_Compatibility_Ideographs

# English Tutorial (Machine translated)

## Installation

Unlike the original OceanFlame, GlyphFlame does not provide an online version.  
You must download the repository and run it locally.

### Requirements
- Bun (please install this in advance)
- glyph-flame-main (repository copy)
- BDF font files (for font replacement)

1. Go to the GlyphFlame repository and click **Code > Download ZIP**.
2. Extract the ZIP file and make sure the contents are directly inside the `glyph-flame-main` folder.
3. Right-click the `glyph-flame-main` folder while holding **Shift**, then select **"Open PowerShell window here"**.
4. Install dependencies:
   ```powershell
   bun install
   ```

## Usage
1. Right-click the `glyph-flame-main` folder while holding `Shift`, then select `Open PowerShell window here`.
2. Start the development server:
   ```powershell
   bun run dev
   ```
   
   Press `O + Enter` to open the tool.
3. Drag and drop your `.img` firmware file onto the center of the screen.
4. Select the font type (SMALL/LARGE) and Font Plane from the left sidebar.
(e.g., `Hiragana`, `Katakana`, `CJK Unified`)

### Loading BDF Font
5. Drag and drop a compatible `.bdf` file onto the main area.
* Small → ~12px BDF (`b12.bdf`)
* Large → ~16px BDF (`b16.bdf`)

Notes:
* First import may not reflect correctly. Switch planes and try again.
* Hiragana/Katakana may appear cut off. Use "Align All Left" to fix.
* Always apply changes before switching planes.

### Editing Glyphs
6. Click a glyph to open the editor.

#### Controls
* Click → Toggle pixel
* Drag → Range selection
* Arrow keys → Move selection
* Esc or "Clear Selection" → Cancel selection

7. Click "Apply This Glyph" to save changes.

### Saving
8. Once all edits are complete, click "Download .img" to export the modified firmware.


## Additional Information

### Recommended BDF Fonts
[efont-unicode-bdf](http://openlab.ring.gr.jp/efont/dist/unicode-bdf/)
(Download the `.tar.bz2` file near the top)

### Recommended Planes for Japanese Text
In my case, I edit the following font planes (both SMALL and LARGE)

* Perform "Align All Left" on:
  * Hiragana
  * Katakana
  * CJK_Unified

* Do not perform "Align All Left" (leave as-is) on:
  * CJK_Compatibility
  * CJK_Extention_A
  * CJK_Compatibility_Ideographs
